# TAREA PRÁCTICA – Programación Tradicional vs POO en Python

Este repositorio contiene dos soluciones desarrolladas para la asignatura de 
Programación: una utilizando el paradigma de **Programación Tradicional** 
y otra empleando **Programación Orientada a Objetos (POO)**.

---

## Objetivo
Comparar ambos paradigmas mediante un programa que solicita las temperaturas 
de los 7 días de la semana y calcula el promedio semanal.
